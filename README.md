# 🖼️ Photo Editor using OpenCV & Streamlit

## 📌 Project Description

This project is a simple web-based photo editor built using Python.
It allows users to upload an image and apply different editing features like resizing, brightness, contrast, filters, and more.

The application uses OpenCV for image processing and Streamlit to create an interactive user interface.

---

## 🛠️ Tools Used

* Python
* OpenCV
* Streamlit
* NumPy
* PIL (Python Imaging Library)

---

## ✨ Features

* Upload an image
* Resize the image
* Adjust brightness and contrast
* Convert image to grayscale
* Apply blur effect
* Apply sharpen effect
* Apply warm filter
* Portrait-style background blur (face detection)
* Edge detection
* Cartoon effect
* Rotate image
* Download the edited image

---

## ▶️ How to Run the Project

### 1. Clone the repository

```
git clone https://github.com/your-username/photo-editor.git
cd photo-editor
```

### 2. Install dependencies

```
pip install streamlit opencv-python numpy pillow
```

### 3. Run the application

```
streamlit run app.py
```

### 4. Open in browser

The app will automatically open in your browser.

---

## 🔄 Project Flow

Upload Image → Apply Adjustments → Apply Filters → View Edited Image → Download Image

---

## 📷 Output

The application displays both the original and edited images and allows users to download the final result.
